🚀 如何分享給朋友（製作可分享的網址）

要讓這個模擬器擁有獨立的網址，你需要將程式碼上傳到一個網頁託管服務。

  1. **使用 GitHub Pages (推薦)**
    • 前往 GitHub.com 並註冊/登入。
    • 建立一個新的倉庫（repository），並給它一個名稱。
    • 將本程式碼區塊中的所有內容，從 `` 到 ``，複製到一個名為 `index.html` 的新檔案中。
    • 將這個檔案上傳到你剛建立的倉庫中。
    • 在倉庫的「Settings」中，找到「Pages」選項，將來源設定為 `main` 分支。
    • GitHub Pages 會自動為你建立一個公開的網址，例如：`https://你的用戶名.github.io/你的倉庫名/`。你就可以分享這個連結了！
  2. **使用 Glitch.com**
    • 前往 Glitch.com。
    • 點擊「New Project」並選擇「Hello Webpage」。
    • 將預設的 `index.html` 內容替換成你複製的程式碼。
    • Glitch 會自動為你的專案生成一個網址，你可以點擊左上角的「Share」來找到它。
  3. **離線分享（不需網址）**
    • 如果只想離線分享，請參照上一版的教學：將程式碼複製並儲存為 `.html` 檔案，然後將檔案直接傳送給你的朋友。
